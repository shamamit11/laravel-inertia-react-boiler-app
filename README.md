## Laravel-React-Inertia Boiler App

- Download Files or Clone Repository
- cd folder_name
- Run: composer install
- Run: yarn install
- Run: yarn run dev
- Setup database and update .env file
- Run Migration: php artisan migrate
- Run: php artisan serve
